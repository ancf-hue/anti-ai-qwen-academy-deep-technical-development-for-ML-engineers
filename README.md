# 🎓 Anti-AI Qwen Academy

**Глубокая техническая разработка для ML-инженеров** — бесплатный офлайн-курс из 25 уроков по созданию математических архитектур нейросетей с нуля.

> 🔗 Демо: [ancf-hue.github.io/anti-ai-qwen-academy](https://ancf-hue.github.io/anti-ai-qwen-academy)

---

## 🎯 Для кого этот курс

- Программисты, желающие перейти в ML-инженерию
- Исследователи, которым нужно понимать низкоуровневую реализацию архитектур
- Разработчики, готовящие модели к Production-деплою

---

## 📚 Структура курса (5 блоков × 5 уроков)

### 🔷 Блок 1: Математический фундамент архитектур
| Урок | Тема |
|------|------|
| 1 | Тензорная алгебра в коде: скаляры, векторы, матрицы, `einsum` |
| 2 | Автоматическое дифференцирование: графы вычислений, Backpropagation |
| 3 | Математика оптимизаторов: SGD, Momentum, Adam, RMSprop |
| 4 | Кастомные функции потерь: Focal Loss, Huber Loss, численная стабильность |
| 5 | Регуляризация: L1/L2, Dropout, ландшафт функции потерь |

### 🔷 Блок 2: Проектирование базовых архитектур вручную
| Урок | Тема |
|------|------|
| 6 | Кастомный слой Linear: инициализация весов, `nn.Parameter` |
| 7 | Функции активации и нормализация: GeLU, LayerNorm, RMSNorm |
| 8 | Сверточные сети (CNN): ResNet-блоки, Depthwise Separable Conv |
| 9 | Рекуррентные сети (RNN/LSTM): ячейки, гейты, затухание градиентов |
| 10 | Пайплайны данных: `Dataset`, `DataLoader`, `collate_fn`, паддинг |

### 🔷 Блок 3: Современные тяжелые архитектуры
| Урок | Тема |
|------|------|
| 11 | Механизм внимания: Scaled Dot-Product, каузальная маска |
| 12 | Multi-Head Attention: параллельные головы, позиционное кодирование |
| 13 | Полная архитектура Transformer: Decoder-Only, Pre-LN, генерация текста |
| 14 | Диффузионные модели: DDPM, U-Net, эмбеддинги времени |
| 15 | Автокодировщики и GAN: VAE, репараметризация, минимаксная игра |

### 🔷 Блок 4: Масштабирование и инженерия LLM
| Урок | Тема |
|------|------|
| 16 | Токенизация текста: BPE, Byte-level, `tiktoken` |
| 17 | Эффективное дообучение (PEFT): LoRA, QLoRA, слияние весов |
| 18 | Оптимизация инференса: квантование (INT8/4), KV-кэш, прунинг |
| 19 | Стратегии генерации: Temperature, Top-P, Repetition Penalty |
| 20 | Выравнивание моделей: RLHF, DPO, лосс предпочтений |

### 🔷 Блок 5: Низкоуровневая оптимизация и Production
| Урок | Тема |
|------|------|
| 21 | Распределённое обучение: DDP, Tensor/Pipeline Parallelism, NCCL |
| 22 | Оптимизация памяти: ZeRO, Mixed Precision, Gradient Checkpointing |
| 23 | CUDA-программирование: Triton, Operator Fusion, ядра на Python |
| 24 | Компиляция моделей: `torch.compile`, TorchInductor, Graph Breaks |
| 25 | Деплой высокой доступности: ONNX, TensorRT, Triton Server, vLLM |
